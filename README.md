# Customer Complaint Analytics using NLP

## Overview
This project analyzes customer complaint narratives using Natural Language Processing (NLP).

## Technologies Used
- Python
- Pandas
- NLTK
- TextBlob
- Matplotlib

## Features
- Sentiment Analysis
- Tokenization
- POS Tagging
- Noun Phrase Extraction
- Keyword Extraction
- Spell Correction
- Translation
- Complaint Trend Visualization

## Output
The project classifies complaints into positive, negative, and neutral categories and visualizes complaint trends across products.
