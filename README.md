# Customer Complaint Analytics using NLP

## Overview
This project analyzes customer complaint narratives using Natural Language Processing (NLP).

## Technologies Used
- Python
- Pandas
- NLTK
- TextBlob
- Matplotlib

## Features
- Sentiment Analysis
- Tokenization
- POS Tagging
- Noun Phrase Extraction
- Keyword Extraction
- Spell Correction
- Translation
- Complaint Trend Visualization

## Dataset

This project uses a Customer Complaints dataset obtained from Kaggle.

Due to file size limitations, the dataset is not included in this repository.

Dataset Source:
(https://www.kaggle.com/datasets/shashwatwork/consume-complaints-dataset-fo-nlp)

Download the dataset from the above link and place the CSV file in the project directory before running the script.

## Output
The project classifies complaints into positive, negative, and neutral categories and visualizes complaint trends across products.
